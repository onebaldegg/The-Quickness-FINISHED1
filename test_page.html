About THE QUICKNESS

THE QUICKNESS is a Chrome extension that allows users to quickly capture screenshots of the visible viewport, add notes, and save them as PDFs with preserved links.

Key features include:

  • Immediate viewport screenshot on extension icon click
  • Note modal with purple background
  • Landscape PDF generation with specific layout
  • Link preservation in screenshots
